打开网页发现有随机数学计算题，刷新几次后会出现新的。

![](https://pic.npiter.de/file/1771552110819_20260220094829921.png)

![](https://pic.npiter.de/file/1771552147118_20260220094857986.png)

用 F12 查看请求头，发现 Cookie 有点特别：

![](https://pic.npiter.de/file/1771552158088_20260220094917367.png)

有 `Timeout=alive`。

![](https://pic.npiter.de/file/1771552214941_20260220095007525.png)

知道这个规律就可以写脚本了：

```python
import re
import requests

url = 'http://117.72.52.127:12222'
headers = {
    'Cookie': 'PHPSESSID=uphbeui9h76bthqeqtejrhbhr6; Timeout=alive'
}

response = requests.get(url, headers=headers)

math_expr = re.search(r"Give me value post about (.*?)=\?", response.text, re.S)
if math_expr:
    value = eval(math_expr.group(1))
    data = {"value": value}
    response = requests.post(url, headers=headers, data=data)
    response.encoding = 'utf-8'
    print(response.text)
else:
    print("Mathematical expression not found in response")
```
